How it works: Greedy Best-First Search always expands the cell with the lowest heuristic value h(n). It uses a priority queue (open list) and a closed set to avoid re-visiting cells. The heuristic is h(n) = |col_n − col_G| + |row_n − row_G| — Manhattan distance to Goal G. It is not guaranteed to find the shortest path, but it is fast.
Algorithm Steps
Each row = one cell expanded from the open list (lowest h first). L/U/R/D = heuristic h of that neighbor  |  W = wall / out-of-bounds   V = already visited (closed)   green = chosen direction  |  Value = h of current cell  |  Next Best = second-lowest h among valid neighbors.
